# Portfolio

My portfolio website - Full Stack Developer & DevOps Engineer exploring Web3.

## Tech Stack

- **Framework:** Next.js 16
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Runtime:** React 19

## Features

- Interactive canvas animation with particle physics
- Keyboard shortcuts for various modes and effects
- Matrix mode theme toggle
- Responsive design

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `G` | Toggle Gravity |
| `T` | Toggle Trails |
| `F` | Freeze Time |
| `E` | Explosion Effect |
| `D` | Debug Overlay |
| `B` | Binary Mode (hold) |
| `Space` | Time Warp (hold) |
